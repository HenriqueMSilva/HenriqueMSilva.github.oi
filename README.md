# Resumo do Projeto

Neste projeto pretendemos criar um jogo, desenvolvido em Javascript, HTML e CSS, 2D,
denominado “Super Spam Shoting StarGuardians SpaceShip”.

O jogo será do estilo “Shot’em up” onde uma pequena nave terá de disparar contra grupos de
naves inimigas e desviar-se de diversos elementos que lhe podem causar dano (vários projeteis, as
próprias naves inimigas, etc…), tendo ainda que aniquilar um “Boss” para passar ao próximo nível.

Será possível apanhar “power ups” ao longo do jogo e comprar novas armas e naves que terão
propriedades diferentes. Isto resultará numa experiência única por cada sessão de jogo.

É nosso objetivo implementar várias mecânicas para manter o jogador entretido e entusiasmado:
por exemplo uma ideia seria implementar “quick time events” para finalizar os bosses, ou para apanhar
certos “power ups” ao longo do jogo, ou ainda permitir a criação/ edição de níveis(?).

Existira um modo de Campanha e um modo Infinito. O de Campanha terá uma leaderboard
organizada por número de níveis que o jogador conclui ou o tempo que demorou a completar este
mesmo modo. O modo Infinito será uma combinação de mecânicas do modo Campanha (por exemplo,
níveis sem “bosses” que vão acelerando ou bosses ao acaso) que também terá uma leaderboard
organizada por tempo de jogo.


# Motivações e Objetivos

O nosso objetivo é criar um jogo de ritmo rápido que teste os reflexos do jogador. Será um jogo
que poderá ser apreciado por indivíduos mais competitivos ou por aqueles mais casuais que apenas
querem passar um tempo de qualidade. Este jogo também tem o objetivo de despertar nostalgia nos
jogadores mais retro visto que este estilo de jogo é muito conhecido e adorado.

apreciar um bom jogo.

- Tópicos a abordar:
 Modo Campanha
 Modo Infinito
 Criação/Edição de Níveis
 Leaderboards
 História
 Ajuda
 Créditos

- Funcionalidades a implementar:
 Editor de Níveis
 Menus
 Leaderboards
 Movimento

- Definições dos Conteúdos:
 Níveis do Modo Campanha
 Algoritmo do Modo Aleatório
 Assets/Sprites dos objetos e personagens
 História
 Soundtrack
 Animações

- Plataforma de Desenvolvimento e Distribuição:
 Iremos usar principalmente o Google Chrome para testar e desenvolver o nosso jogo,
sendo especializado para o mesmo, o que permite assim uma grande portabilidade e
acessibilidade.
 Vamos usar o Sublime Text como editor de texto e o GIMP como editor de Imagem.
 E ainda o Git Bash será usado para manter o trabalho sincronizado entre os elementos
do grupo.
